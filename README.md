# @koppajs/language-server

Language Server Protocol implementation for KoppaJS.

## Scope

This package exposes the shared KoppaJS language layer over LSP:

- semantic completions
- diagnostics
- hover, definition, references, rename
- document and workspace symbols
- quick fixes
